# AI Internship Screening System
A Machine Learning-based web application that predicts whether a candidate is likely to be selected for an internship.

## 🚀 Features
- Logistic Regression ML model
- Real-time prediction using Flask
- Probability confidence score
- Clean UI interface

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- HTML/CSS
- Git & GitHub

## 📂 Project Structure
- model.py → Model training
- app.py → Flask backend
- templates/ → HTML frontend
- model.pkl → Trained model

## 📌 How to Run
1. Create virtual environment
2. Install dependencies
3. Run `python app.py`
4. Open http://127.0.0.1:5000
